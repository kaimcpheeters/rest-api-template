# REST API TEMPLATE

## Getting started

To start project, run:

```
docker-compose up
```

The API will then be available at http://127.0.0.1:8000

Create a user:

http://127.0.0.1:8000/api/user/create/

Create a token:

http://127.0.0.1:8000/api/user/token/

Manage user settings:

http://127.0.0.1:8000/api/user/me/
